# E-commerce API

1. Clone repository
2. Execute command
``` 
yarn install 
```
3. Set up your __.env__ file based on __.env.template__
4. Start database
```
docker-compose up -d
```
5. Start project 
```
yarn start:dev
```
6. Execute seed
```
http://localhost:3000/api/seed
```
